
# adbot node

```bash
# adbot  node
NAME:
   adbot node - node management

USAGE:
   adbot node command [command options] [arguments...]

COMMANDS:
     ls        list all of nodes
     inspect   inspect details of a node
     stats     watch live stream of node stats
     terminal  open a terminal on one give node
     exec      exec command in a specified node
     watch     watch events of a node
     label     manage node labels
     close     close a specified node once
```
